# WAP_milestone-project

## Project Overview

Crypto Tracker is a web application that allows users to track and explore cryptocurrency market data in real time. The application fetches live cryptocurrency data from a public API and displays it in a user-friendly interface. Users can search, filter, and sort cryptocurrencies based on different parameters such as price, market cap, and name.

This project demonstrates API integration, JavaScript array higher-order functions, and responsive UI development.

---

## Features

* Fetch real-time cryptocurrency data from API
* Display cryptocurrencies in cards/table format
* Search cryptocurrencies by name
* Filter cryptocurrencies based on price or market cap
* Sort cryptocurrencies (price, name, market cap)
* Responsive design for mobile, tablet, and desktop
* Dark mode / Light mode toggle
* Pagination
* Loading indicator

---

## API Used

CoinGecko API
https://api.coingecko.com/api/v3/coins/markets

This API provides real-time data for cryptocurrencies including price, market cap, volume, and rankings.
